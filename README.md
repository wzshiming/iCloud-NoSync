## Install

``` bash
mkdir -p $HOME/Library/Services/NoSync.workflow && \
  curl -L https://github.com/wzshiming/iCloud-NoSync/archive/refs/heads/master.zip | \
  tar -xf - --strip-components=2 -C $HOME/Library/Services/NoSync.workflow
```

## Uninstall

``` bash
rm -rf $HOME/Library/Services/NoSync.workflow
```
